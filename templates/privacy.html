Privacy Protocol

// UPDATED: FEB 2026
TL;DR: AES-256 Encrypted. Zero Logging. Automatic self-destruction of data.

01. Collected Info

  • GitHub Tokens (Encrypted)
  • Repository Endpoints
  • Optional Access Passwords
  • Expiration Metadata

02. Data Processing

Sensitive data is processed using AES-256 encryption. Encryption keys are generated per-share via PBKDF2. We never see your plain-text tokens.

03. Automatic Cleanup

Links are set to self-destruct (1-30 days). Once the timer hits zero, every database entry, token, and reference is permanently purged.

04. We DON'T Do

  • Store your source code
  • Sell tokens to 3rd parties
  • Track your browsing habits
  • Keep logs of your passwords

05. Third Parties

We only interface with GitHub API, MongoDB Atlas (storage), and optionally Gemini AI for README help.

06. Your Rights

You have the right to manually delete any share link instantly, or request information about your active encrypted sessions.

07. Cookies & Sessions

We use essential session cookies only to keep you logged in or save encrypted tokens temporarily. Data is cleared after 24 hours of inactivity.
